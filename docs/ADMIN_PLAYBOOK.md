# Admin Playbook

## Daily Operations

### Morning Routine (15 min)
1. Check `/admin/revenue` dashboard
2. Review overnight signups/cancellations
3. Check Discord for urgent issues
4. Review support inbox

### Midday (30 min)
1. Respond to support tickets
2. Approve community posts (if moderated)
3. Check affiliate performance
4. Review application queue

### Evening (15 min)
1. Check Stripe for payment issues
2. Review next day's scheduled content
3. Note any anomalies for follow-up

## Support Tickets

### Response Times
- **General**: Within 24 hours
- **Billing**: Within 12 hours
- **Technical**: Within 4 hours
- **Elite**: Within 2 hours

### Common Issues

**Can't access content**
1. Check subscription status
2. Verify login credentials
3. Clear browser cache
4. Escalate if unresolved

**Billing question**
1. Check Stripe dashboard
2. Review payment history
3. Explain charges
4. Process refund if applicable

**Cancel subscription**
1. Offer to resolve issue
2. Process cancellation
3. Send exit survey
4. Add to win-back list

**Upgrade question**
1. Explain ELITE benefits
2. Offer trial/discount
3. Process upgrade
4. Welcome to ELITE

## Crisis Management

### Website Down
1. Check Vercel status
2. Check Supabase status
3. Check Stripe status
4. Post status update
5. Communicate with members

### Payment Issues
1. Check Stripe dashboard
2. Identify affected members
3. Send proactive communication
4. Offer temporary access
5. Process refunds if needed

### Community Incident
1. Assess severity
2. Remove offending content
3. Warn/suspend member
4. Document incident
5. Update guidelines if needed

### Legal Issue
1. Document everything
2. Contact legal counsel
3. Do not admit liability
4. Follow legal advice
5. Update policies

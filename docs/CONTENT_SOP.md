# Content Publishing SOP

## Content Types

### 1. Cornerstone Articles (2,000+ words)
**Purpose**: SEO traffic, authority building
**Frequency**: 2 per month
**Examples**:
- Country/visa guides
- Cost of living comparisons
- Tax optimization strategies

### 2. City Guides (1,500+ words)
**Purpose**: Member value, affiliate revenue
**Frequency**: 4 per month
**Template**:
- Why this city
- Cost breakdown
- Visa requirements
- Lifestyle overview
- Risks & considerations
- Member perks
- Affiliate CTAs

### 3. Long-Tail Articles (800-1,200 words)
**Purpose**: SEO, answering specific questions
**Frequency**: 4 per month
**Examples**:
- "Lisbon vs Porto"
- "Bangkok neighborhoods"
- "Mexico City safety"

### 4. Email Newsletter
**Purpose**: Engagement, retention, upsells
**Frequency**: Weekly
**Sections**:
- Community highlight
- New content
- Featured deal
- Upcoming events
- Member question

### 5. Social Media
**Purpose**: Brand awareness, community building
**Frequency**: Daily
**Platforms**: Twitter/X, Instagram, LinkedIn

## Publishing Workflow

### Step 1: Research (Day 1-2)
- [ ] Keyword research (Ahrefs/Semrush)
- [ ] Competitor analysis
- [ ] Member questions review
- [ ] Data collection

### Step 2: Outline (Day 2)
- [ ] H2/H3 structure
- [ ] Key points to cover
- [ ] Affiliate link placements
- [ ] Internal link opportunities

### Step 3: Write (Day 3-4)
- [ ] First draft
- [ ] Include personal anecdotes
- [ ] Add data/statistics
- [ ] Write compelling intro

### Step 4: Edit (Day 5)
- [ ] Grammar check (Grammarly)
- [ ] Readability (Hemingway)
- [ ] SEO optimization (Yoast)
- [ ] Fact-check claims

### Step 5: Optimize (Day 5)
- [ ] Meta title (60 chars max)
- [ ] Meta description (160 chars max)
- [ ] Header tags (H1, H2, H3)
- [ ] Image alt text
- [ ] Internal links (3-5)
- [ ] External links (2-3)
- [ ] Affiliate links

### Step 6: Publish (Day 6)
- [ ] Upload to CMS
- [ ] Add featured image
- [ ] Set category/tags
- [ ] Schedule publish time (Tuesday 9am EST)

### Step 7: Promote (Day 6-7)
- [ ] Share on social media
- [ ] Email newsletter
- [ ] Community announcement
- [ ] Repurpose for other formats

## SEO Checklist

### On-Page SEO
- [ ] Target keyword in H1
- [ ] Target keyword in first 100 words
- [ ] Keyword density 1-2%
- [ ] LSI keywords throughout
- [ ] URL includes target keyword
- [ ] Images optimized (WebP, compressed)
- [ ] Page load speed <3 seconds
- [ ] Mobile responsive

### Technical SEO
- [ ] XML sitemap submitted
- [ ] Robots.txt configured
- [ ] Schema markup added
- [ ] Canonical tags set
- [ ] SSL certificate active
- [ ] No broken links

### Content Quality
- [ ] Original content (no plagiarism)
- [ ] Comprehensive coverage
- [ ] Up-to-date information
- [ ] Credible sources cited
- [ ] Actionable advice
- [ ] Engaging writing style

## Writing Guidelines

### Tone
- Confident but not arrogant
- Informative but not dry
- Premium but accessible
- Masculine but respectful

### Style
- Short paragraphs (2-3 sentences)
- Bullet points for lists
- Bold key takeaways
- Use "you" and "we"
- Include real examples

### Prohibited
- Hype or exaggeration
- Unverified claims
- Political opinions
- Dating advice that's crude
- Income guarantees

---

## Affiliate Link Placement

### Best Practices
- Natural integration
- Contextual relevance
- Clear disclosure
- Trackable URLs
- Test all links

### Placement Rules
- Max 3 affiliate links per 1,000 words
- First link in first 25% of content
- CTA at end of article
- Sidebar widgets on guides

### Disclosure
```
**Affiliate Disclosure**: This article contains affiliate links. 
We may earn a commission if you use our links, at no extra cost 
to you. We only recommend services we trust.
```

## Repurposing Content

### Article → Email
- Extract key points
- Add personal story
- Include CTA

### Article → Social
- Pull quotes
- Create thread (Twitter)
- Design carousel (Instagram)
- Write summary (LinkedIn)

### Article → Video
- Script from article
- 5-10 minute length
- Screen share + face
- YouTube + TikTok

## Performance Tracking

### Metrics to Track
- Organic traffic (Google Analytics)
- Keyword rankings (Ahrefs)
- Engagement time
- Bounce rate
- Affiliate clicks
- Conversion rate

### Review Schedule
- **Weekly**: Traffic, rankings
- **Monthly**: Top/bottom performers
- **Quarterly**: Content audit, update old posts
